# UE5 办公室房间外壳场景布局规格

## 概述

本文档定义 WhyBuddy UE5 场景中房间外壳（墙体、地板、天花板）的精确尺寸与放置规格。所有数据均从 Three.js `OfficeRoom` 组件提取，并按 UE5 坐标系与单位系统转换。

---

## 1. 坐标系映射

### 1.1 坐标轴对照

| 轴 | Three.js（Y-up） | UE5（Z-up） | 说明 |
| -- | ---------------- | ----------- | ---- |
| 水平横向 | X | X | 方向一致，左右 |
| 垂直向上 | Y | Z | Three.js Y → UE5 Z |
| 水平纵深 | Z | Y | Three.js Z → UE5 -Y（注意取反） |

### 1.2 位置转换公式

```
UE5_X =  ThreeJS_X × 100        (cm)
UE5_Y =  ThreeJS_Z × -100       (cm，Z 轴取反)
UE5_Z =  ThreeJS_Y × 100        (cm)
```

### 1.3 尺寸转换公式

```
UE5_SizeX = ThreeJS_SizeX × 100  (cm)
UE5_SizeY = ThreeJS_SizeZ × 100  (cm)
UE5_SizeZ = ThreeJS_SizeY × 100  (cm)
```

> **注意**：Three.js 使用米（m）为单位，UE5 使用厘米（cm）为单位，统一缩放因子 = **100**。

## 7. 天花板构建建议

Three.js 版本中没有显式的天花板网格（开放顶部），但在 UE5 中建议添加天花板以实现完整的室内光照效果。

### 7.1 推荐天花板规格

| 属性 | 值 | 说明 |
| ---- | -- | ---- |
| 位置 (UE5) | (0, 0, 300) | 墙体顶部 |
| 尺寸 (UE5) | 1542 × 998 × 10 cm | 与后墙宽度和侧墙深度匹配 |
| 材质颜色 | #F0E8DD 或 #FFFFFF | 白色或暖白色 |
| 粗糙度 | 0.95 | 哑光效果 |

### 7.2 天花板灯光开口

建议在天花板上预留灯光开口或使用 Rect Light 模拟顶灯效果：

- 中央区域可放置 2-4 个 Rect Light 模拟日光灯
- 天花板材质可使用略带暖色的白色，与墙面色调协调

## 9. UE5 搭建步骤摘要

1. **创建地板**：在 `(0, 0, 0)` 放置 1800×1400 的平面，应用木地板材质。
2. **放置后墙**：在 `(0, 490, 150)` 放置 1542×18×300 的 BSP 或 Static Mesh。
3. **放置左墙**：在 `(-780, 0, 150)` 放置 18×998×300 的 BSP 或 Static Mesh。
4. **放置右墙**：在 `(780, 0, 150)` 放置 18×998×300 的 BSP 或 Static Mesh。
5. **添加踢脚线**：沿三面墙底部放置踢脚线模型或 BSP。
6. **放置墙角**：在左后角和右后角放置对应的墙角模型。
7. **放置门洞**：在右墙 `(765, 290, 0)` 处放置宽门洞模型。
8. **铺设装饰地板砖**：按照第 4.3 节的位置表放置 Kenney 地板模型。
9. **添加天花板**：在 Z=300 处放置天花板平面。
10. **调整材质**：为所有元素分配对应的 PBR 材质实例。
