# UE5 办公室工位区场景布局规格

## 概述

本文档定义 WhyBuddy UE5 场景中 4 组工位区（Pod A-D）及 CEO 办公桌的精确放置规格。所有数据均从 Three.js `OfficeRoom` 组件提取，并按 UE5 坐标系与单位系统转换。

---

## 1. 坐标系映射（与 `scene-layout-room-shell.md` 一致）

### 1.1 位置转换公式

```
UE5_X =  ThreeJS_X × 100        (cm)
UE5_Y =  ThreeJS_Z × -100       (cm，Z 轴取反)
UE5_Z =  ThreeJS_Y × 100        (cm)
```

### 1.2 旋转转换公式

```
UE5_Yaw = -ThreeJS_RotationY × (180 / π)   (度)
```

> **注意**：Three.js 使用米（m）为单位，UE5 使用厘米（cm）为单位，统一缩放因子 = **100**。

## 8. UE5 搭建步骤摘要

### 8.1 推荐搭建顺序

1. **放置 CEO 办公桌**：在 `(0, 315, 0)` 放置 DesktopDesk 组合件（withLamp），铺设圆形地毯。
2. **搭建 Pod A（左后）**：先铺地毯，再放置台式桌（compact）、笔记本桌、休闲椅和矮书柜。
3. **搭建 Pod B（右后）**：先铺地毯，再放置笔记本桌、会议桌组合和边桌。
4. **搭建 Pod C（左前）**：先铺地毯，再放置会议桌组合、笔记本桌、休闲椅和矮书柜。
5. **搭建 Pod D（右前）**：先铺地毯，再放置茶几、两把休闲椅、笔记本桌和矮书柜。

### 8.2 组合件放置建议

- **DesktopDesk / LaptopDesk**：建议在 UE5 中创建 Blueprint Actor，将桌、椅、显示器/笔记本、键盘、鼠标等子物体作为组件，统一管理偏移与旋转。
- **MeetingSet**：同样建议创建 Blueprint Actor，包含圆桌和 3 把椅子。
- **StorageColumn (low)**：矮书柜 + 书本组合，书本位于 Y=50cm 处。

### 8.3 注意事项

- 所有 `centerXZ` 标记的模型在 Three.js 中已做中心对齐，UE5 中需确保 Pivot 在底面中心。
- 地毯的 Z 坐标为 1cm（UE5），略高于地面以避免 Z-fighting。
- compact 模式的 DesktopDesk 椅子偏移为 72cm（而非标准 82cm），键盘偏移也有差异。
