# UE5 办公室休息区与装饰元素场景布局规格

## 概述

本文档定义 WhyBuddy UE5 场景中休息区（沙发 + 茶几 + 休闲椅）、装饰植物和建筑装饰元素（衣帽架、落地灯、壁灯）的精确放置规格。所有数据均从 Three.js `OfficeRoom` 组件提取，并按 UE5 坐标系与单位系统转换。

---

## 1. 坐标系映射（与 `scene-layout-room-shell.md` 一致）

### 1.1 位置转换公式

```
UE5_X =  ThreeJS_X × 100        (cm)
UE5_Y =  ThreeJS_Z × -100       (cm，Z 轴取反)
UE5_Z =  ThreeJS_Y × 100        (cm)
```

### 1.2 旋转转换公式

```
UE5_Yaw = -ThreeJS_RotationY × (180 / π)   (度)
```

### 1.3 局部偏移转换公式

```
UE5_LocalX = ThreeJS_LocalX × 100
UE5_LocalY = ThreeJS_LocalZ × -100
UE5_LocalZ = ThreeJS_LocalY × 100
```

> **注意**：Three.js 使用米（m）为单位，UE5 使用厘米（cm）为单位，统一缩放因子 = **100**。

## 2. 休息区总览

休息区位于房间前方中央（Three.js Z 正方向 / UE5 Y 负方向），以一组长沙发为核心，围绕茶几和休闲椅形成开放式休闲空间。

### 2.1 区域组成

| 区域 | 包含物体 | 说明 |
| ---- | -------- | ---- |
| 主休息组（LoungeArea） | 长沙发、休闲椅、方形茶几、边桌、桌灯 | 以 LoungeArea 组为父级 |
| 中央补充家具 | 圆形茶几、2 把独立休闲椅 | 散布在主休息组周围 |

## 8. UE5 搭建步骤摘要

### 8.1 推荐搭建顺序

1. **放置主休息组**：在 `(20, -410, 0)` 创建空 Actor 或 Blueprint 作为组根节点，依次添加长沙发、休闲椅、方形茶几、边桌和桌灯。
2. **放置中央补充家具**：在世界坐标放置圆形茶几和 2 把独立休闲椅。
3. **放置装饰植物**：按照第 5 节的位置表，在房间四角和前方两侧放置 6 盆植物。
4. **放置建筑装饰**：放置衣帽架、落地灯和壁灯。
5. **配置灯光**：为落地灯和壁灯添加 Point Light 组件，按照第 6 节的参数设置颜色、强度和衰减距离。

### 8.2 注意事项

- 长沙发旋转 180°（Yaw=-180°），使其面朝房间中央。
- 桌灯放置在边桌上方，Z 坐标为 70cm（边桌高度约 0.7m）。
- 植物缩放值直接应用于 UE5 的 Uniform Scale。
- 壁灯挂在后墙上，Z 坐标为 108cm（离地约 1.08m）。
- 落地灯和壁灯的点光源位置与灯体位置略有偏移，模拟灯头发光位置。
