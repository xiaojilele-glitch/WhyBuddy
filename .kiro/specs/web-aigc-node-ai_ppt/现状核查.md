## 当前可认定完成的范围

当前可以把 spec 顶层 4 个任务项全部勾选，但成立前提是：

- 这里的“AI PPT 生成”是最小版页纲闭环，不是完整的二进制 `.pptx` 生产链路。
- 当前 artifact 是可下载的 `.ppt.json` 结构化产物，用于承载演示文稿内容与页面结构。
- 当前 route 与 adapter 已可被主线接入，但未在本次范围内修改 `server/index.ts` 或 runtime 主线注册逻辑。

## 仍需主线处理的事项

以下事项不影响当前 spec 在“最小闭环”意义上收口，但如果要让主流程自动可达，仍需主代理统一接线：

- 将 `server/routes/ai-ppt.ts` 注册进服务入口
- 将 `ai_ppt` 节点 adapter 接入主线 runtime/节点注册表
- 若产品需要真实下载 `.pptx` 文件，则需后续补充二进制渲染器或外部生成服务
